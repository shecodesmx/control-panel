# 🧩 Control_Panel

Panel de control interactivo creado con **HTML, CSS y JavaScript**, con botones, switches y elementos dinámicos de interfaz.

---

## 🚀 Tecnologías

- HTML5
- CSS3
- JavaScript

---

## ✨ Características

- 🧩 Panel de control UI moderno
- 🔘 Botones interactivos
- 🎛️ Switches y toggles
- ⚡ Diseño ligero
- 📱 Responsive

---

## 📂 Estructura del proyecto

```
control-panel/
│
├── control_Panel.html
├── style.css
├── script.js
```

## ▶️ Cómo usar

1. Descarga el proyecto
2. Abre `control_Panel.html` en tu navegador
